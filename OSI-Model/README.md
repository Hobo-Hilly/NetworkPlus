OSI model

Layer 7 - Application                         
- End User layer
- HTTP, FTP, IRC, SSH, DNS

Layer 6 - Presentation
- Syntax layer
- SSL, SSH, IMAP, FTP, MPEG, JPEG

Layer 5 - Session
- Synch & send to port
- API's, Sockets, WinSock

Layer 4 - Transport
- End-to-end connections
- TCP,UDP

Layer 3 – Network
- Packets
- IP,ICMP,IPsec,IGMP

Layer 2 – Data Link 
- Frames
- Ethernet, PPP, Switch, Bridge

Layer 1 – Physical Structure
Coax, Fiber, Wireless, Hubs, Repeaters
